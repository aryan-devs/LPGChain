# LPGChain

## Blockchain-Based Transparent LPG Distribution & Fraud Detection System

### Problem
LPG black selling and fake shortages happen because there is no transparency in stock allocation and delivery.

### Solution
LPGChain tracks LPG allocation, distributor stock, customer booking, and deliveries with a blockchain-like transaction log.

### Modules
- Admin Module
- Distributor Module
- Customer Module
- Fraud Detection Module
- Transaction Log

### Team Goal
Build a working prototype in 5 days.
